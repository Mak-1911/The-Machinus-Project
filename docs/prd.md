## 📄 Product Requirements Document

### Product: Machinus

Version: v0.1 (Structured AI Cloud Computer)

### 1. Product Vision

> Machinus is a personal, AI-controlled cloud computer that: 
> 
> 1. Runs on a VPS (self-hosted or managed)
> 2. Accepts natural language instructions
> 3. Executes infrastructure and automation tasks.
> 4. Maintains persistent system state
> 5. Provides transparent, inspectable execution
> 
> 
> 
> Machinus is not just an AI assistant.
> 
> It is an AI-native control plane for a personal cloud machine.

## 2. Core Value Proposition

Machinus provides:  
⟫ AI-assisted machine controlled  
⟫ Deterministic execution     
⟫ Transparent plan preview    
⟫ Persistent system registry      
⟫ Multi-interface control (Web, CLI, messaging later)     
⟫ Self-hosted or managed deployment   

> Positioning:    
“Your cloud machine — with a structured AI brain.”
